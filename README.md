# Financial accounting
Application for personal finance accounting.

### The list of the main features of the program:
- Creation expenses
- Reports by expenses
- dark/light theme
- Russian/English localization

## Stack
### Frontend:
- NextJS
- Redux Saga
- Axios
- D3
- Material UI
- yup/formik
- Typescript

### Backend
- NestJS
- Typescript
- PostgreSQL
- TypeORM

## Instalation
### Frontend
```shell
 cd client
 npm install
```
### Backend
```shell
 cd server
 npm install
```
